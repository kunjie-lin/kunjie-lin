# Hi, I'm Kunjie Lin 👋

I'm a second-year M.Eng. student at **Xiamen University**, advised by [Prof. Xinghao Ding](https://scholar.google.com/citations?user=k5hVBfMAAAAJ). My research sits at the intersection of **LLM post-training** and **agentic AI**.

## 🔬 Research Interests

- **LLM Post-Training** — data synthesis, reward modeling, Agentic RL
- **Multi-turn Interactive Agents** — tool use, environment grounding, long-horizon planning
- **Multimodal Reasoning** — vision-language models, chain-of-thought with visual feedback
